Conteúdo Textual

Usam-se elementos HTML de conteúdo textual para se organizar blocos ou seções de conteúdo postos entre as tags de abertura <body> e fechamento </body>. Importantes para acessibilidade e SEO, esses elementos identificam o propósito ou estrutura do conteúdo.

O conteúdo textual é tudo aquilo que é a página.

As principais tags utilizadas são:

<div> - são caixas vazias que são utilizadas para inserir conteúdo a fim de serem estilizadas depois, via CSS.

Aguardando estilo CSS.

<dl> e <dt> - trabalham juntas e engloba uma lista de pares de termos e descrições. Um uso comum para este elemento pe para implementar glossários ou exibir metadados, com lista de pares chave e valor.

tag <hr> - Linha Horizontal

Representa uma quebra temática entre elementos de nível de parágrafo, por exemplo, uma mudança da cena de uma história ou uma mudança de tema com uma seção. Em versões anteriores do HTML, hr representava apenas uma linha horizontal. Agora ela também pode ser apenas uma linha horizontal, porém também está definida em termos semânticos, em vez de termos de apresentação.

<li> - Itens de Lista

É usado para representar um item de lista. Trabalha juntamente com o <ol> e o <ul>
