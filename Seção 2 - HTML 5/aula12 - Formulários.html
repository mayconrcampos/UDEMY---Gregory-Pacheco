Formulários Web - Trabalhando com dados do usuário

Este módulo fornece uma série de artigos que o ajudarão a dominar o essencial das formas web. Os formulários da Web são uma ferramenta muito poderosa para interagir com os usuários — mais comumente eles são usados para coletar dados dos usuários ou permitir que eles controlem uma interface de usuário. No entanto, por razões históricas e técnicas, nem sempre é óbvio como usá-los em todo o seu potencial. Nos artigos listados abaixo, abordaremos todos os aspectos essenciais dos formulários da Web, incluindo marcar sua estrutura HTML, controlar os controles de formulário de estilo, validar dados de formulário e enviar dados ao servidor.

Temos os seguintes métodos que se comportam de maneiras diferentes.

GET, POST, PUT, DELETE

GET

Read - Ler dados do servidor

Este método serve pra enviar informações ao servidor através da URL.

POST

Create - Cria novos dados no servidor.

Este método envia as informações ao servidor sem utilizar a URL, podendo ser assim mais seguro.

PUT

UPDATE - Atualiza dados no servidor.

Este método se comporta de forma semelhante a ambos.

DELETE

DELETE - Exclui dados do servidor.

Este também passa informações via URL.
