Elementos Interativos e Depricated

Elemento Details

Elemento details: é usado como uma ferramenta de onde o usuário irá obter informações adicionais.

Exemplo:

Alguns detalhes

Mais informações sobre os detalhes.

Mais um pouquinho de informações.

Elemento Dialog

O elemento menu representa um grupo de comandos que um usuário pode executar ou ativar.

Isso inclui os menus de lista, que podem aparecer na parte superior de uma tela, bem como menus de contexto, como aqueles que podem aparecer sob um botão depois de clicado.

Vamos ver o exemplo abaixo:

Clique com o botão direito do mouse para ver o menu.
Explodir Computador Deletar todos os Arquivos
Acabar com tudo Estou meio depre hoje

Este elemento não funciona em navegadores da base chromium, somente vindo a funcionar no firefox.
