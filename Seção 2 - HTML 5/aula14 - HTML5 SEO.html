O que é?

Search Engine Optimization

É uma área de estudo voltada a otimização das paginas web para os motores de busca. Mas porquê isso é importante?

Se você tiver um site bem indexado, ele será dos primeiros a aparecerem nos motores de busca.

Os sites que tiverem o melhor SEO, serão os que vão estar no topo das buscas.

Exemplo: Quando procuramos pela palavra chave "computador" no Bing, as primeiras ocorrências que apareceram foram

Logo, os sites mais indexados para este caso foram o Magazine Luiza. Sendo que acima destes encontravam-se os anuncios pagos.

Temos dois tipos de indexados, as buscas pagas e orgânicas.

Pagas são aqueles que literalmente pagam para aparecerem no topo das pesquisas.

Orgânicas são aqueles que aparecem por terem o melhor SEO, logo se diferenciam diante os demais sites.
